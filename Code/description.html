게임 설명

  1. ‘Ppeokkeum Ppeokkem’은 뻐끔이를 공격하는 상어로부터 먹이를 모두 먹어 탈출하는 게임이다.
  2. 상어가 어디로 움직이지 모르고 스테이지가 지날수록 움직임이 빨라진다.
  3. 뻐끔이가 상어와 만나서 닿으면 라이프가 하나씩 줄어들고 시작지점으로 돌아가게 된다.
  4. 게임을 진행할 때마다 라운드별로 통과시간, 남은 라이프 수에 따라 추가 점수가 주어진다.
  5. 각 라운드에서 라이프의 개수를 많이 남기고, 퀘스트를 모두 달성하여 최종 점수에서 높은 점수를 받아 ‘Ppeokkem Ppeokkem’의 1인자가 되세요!
